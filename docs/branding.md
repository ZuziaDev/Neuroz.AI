# Branding

This project uses the `Neuroz.AI` banner asset stored at:

- `assets/neuroz-ai-banner.svg`

## Intended Usage

- GitHub repository header section in `README.md`
- Documentation and preview assets
- Lightweight vector-first branding for dark or light backgrounds

## Notes

- The banner is an SVG asset, so it stays crisp in GitHub and documentation views.
- If a raster export is needed later, this SVG can be converted into `png` or `webp`.
